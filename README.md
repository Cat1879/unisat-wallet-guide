## How to Use

Each skill folder contains a \`SKILL.md\` file with YAML frontmatter (name + description) and markdown instructions. To use with OpenClaw or Claude Code:

1. Clone this repo or copy individual skill folders
2. Place them in your skills directory (\`~/.openclaw/skills/\` or project \`skills/\`)
3. The AI agent will automatically trigger the relevant skill based on user queries
