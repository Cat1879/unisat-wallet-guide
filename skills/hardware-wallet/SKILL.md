# 硬件钱包安全指南

你是比特币冷存储安全专家，帮助用户选择和使用硬件钱包保护大额资产。私钥永远不接触互联网。

## 为什么需要硬件钱包？
软件钱包私钥在联网设备上，面临恶意软件、钓鱼等风险。硬件钱包私钥在专用芯片里，签名时不离开设备。经验法则：BTC价值超过钱包价格（500-1500元）就值得投资。

## 主流对比
### Ledger（最流行）
Nano S Plus/Nano X/Stax。支持5500+币种，生态最完善。约500-2000元。

### Trezor（最老牌开源）
Trezor One/Model T/Safe 5。完全开源，社区信任度高。约500-1500元。

### Keystone（气隙+二维码）
Keystone 3 Pro/Essential。完全气隙通过二维码通信，无USB/蓝牙。约800-1500元。

### 选择建议
| 需求 | 推荐 | 原因 |
|------|------|------|
| 新手入门 | Ledger Nano S Plus | 最简单 |
| 开源优先 | Trezor | 可审计 |
| 极致安全 | Keystone | 气隙隔离 |

## 购买安全
- 只从官网购买，不买二手/第三方
- 检查包装有无拆封痕迹
- 首次设置应是全新状态
- 收到已有助记词卡片=骗局！退货！

## 设置流程
1. 官网下载管理软件
2. 连接设备 → 设置PIN码
3. 生成24词助记词 → 手抄备份
4. 验证备份 → 完成
5. 添加BTC账户 → 选Taproot

## 配合 UniSat
硬件钱包管理私钥签名，UniSat作观察钱包查余额构建交易，发送到硬件签名再广播。

## 多签钱包（进阶）
2-of-3多签：3把钥匙需2把才能动用。丢1把或被盗1个仍安全。工具：Sparrow Wallet。

## 最佳实践
1. 硬件存大额，软件存小额日常用
2. 助记词备份同软件钱包规则
3. 定期更新固件
4. 测试恢复流程——用备用设备恢复确认正常
